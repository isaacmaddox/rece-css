Rece-css

A basic CSS library

Project is still in early development, expect bugs such as unused links

What is Rece-css?

This is a small CSS library meant to provide some basic styles for developers. If you don't like messing with CSS or just don't have the time, this is the place for you!

This project was inspired by water.css. I highly recommend checking that project out too.

Rece-css?

It's a corny name, but it's based of the idea that this library does not rely on classes, so it's like recess.

Usage

To use Rece-css in your project, you can download the source files, or include this tag in the head of your project:

<link rel="stylesheet" href="https://raw.githubusercontent.com/isaacmaddox/rece-css/main/src/sass/styles.css">

One of the main features of this library is that no classes are necessary. Some class options are available so as to avoid restricting developers to my personal preferences. For a full documentation of this project, check out the docs.

FAQ

Can I use this in my own project?

Yes, of course! Attribution is appreciated but not required. If you happen to use this library, feel free to send me a link to your page at isaac@isaacmaddox.net!

Obviously, you can also tweak things in the stylesheet to fit your personal preference. That being said, please avoid tweaking minor things and then claiming the project as your own.

I found a bug! How do I let you know?

If you find a bug with this library, feel free to email me at isaac@isaacmaddox.net or fork the Github repo and patch the bug. I'm also always open to feedback; if you find something that you think could be improved upon, please let me know!
